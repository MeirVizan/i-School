About The System.

i-School is a face recognition based attendance system. For now uses the USB cameras connected to the server's host, as a simulator of IP cameras that installed in the entrance of every classroom and its exit. The system tracks the attendance percentage of each student per lesson. Each camera's IP (now USB connection number) stored in db by admin permitted user, each camera in DB has a column 'status' (exit\entry) and 'classroom number'.

The algorithms used for face recognition and face detection are of open-cv python library. Much of the instructions were taken from OpenCV Face Recognition by Adrian Rosebrock including the face aligner algorithm PyimagesearchThe model for face detection is caffe model.
Each student has a folder (within dataset folder) on his name and ID number. The folder contains the students facial images. The dataset should contain an unknown folder which should store the same amount of unknown faces as the amount that stored in each known student directory.

Attendance monitoring will be executed manually by the teacher, using start/end class button on real time.Students are tracked as long as the cameras are on, regardless of an active class. The start/end class is for counting the time of each student been attendant in the classroom along the lesson.
