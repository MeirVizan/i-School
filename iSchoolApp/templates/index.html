Smart Attendance™

Face Recognition system to reduce attendance time

The Challenge.

  • Finding / Creating a Reliable Face Recognition Algorithm Face Recognition from a Limited Database of Data for Each Student
  • Adding a new face and training the model without running on all existing faces
  • Face recognition in non-optimal situations such as face tilted right, left, up, down and covered face parts

The Solution.

SmartIdentification™ is an AI powered face recognition solution which uses Computer Vision and Machine Learning algorithms to mark the attendance of students in school.

The system will work on face recognition where each student in the class will be photographed and their details will be stored in a server. The teacher can then record the attendance by just clicking some pictures of the classroom. The system will recognize the faces and verify the presence or absence of each student.
