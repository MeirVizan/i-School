Smart Attendance™

Face Recognition system to reduce attendance time

The Challenge.

    Algorithm
  • Finding an existing reliable Face Recognition Algorithm from an open source code.
  • An algorithm which requires a reasonable amount of face images per student.
  • Models of face detection from multiple position such as frontal right and left face profiles.
    • System
  • Reduce attendance time.
  • Constant tracking student attendance in all classrooms at any given moment.
  • Tracking the time each student has been present in the classroom during the lesson.

The Solution.

i-School is an AI powered face recognition solution which uses Computer Vision and Machine Learning algorithms to mark the attendance of students in school.

How it Works.

Using a range of algorithms that cv2.dnn provides for face detection, embedding extraction and recognition in real time. The system will work on face recognition where each student in the class will be photographed and their details will be stored in a server. The teacher can then start attendance by clicking start/end class button.
